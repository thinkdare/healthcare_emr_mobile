Implementation notes

  • ProviderDashboardScreen drawer: add an if (auth.staffType == 'admin') block rendering the Admin section between the Clinical and Account groups. No other changes to existing drawer items.
  • MoreScreen: same guard, new CupertinoListSection.insetGrouped block with header "Admin", inserted between the Clinical and Account sections.
  • All four admin entries navigate via Navigator.push — no new named routes needed.
  • Non-admin users see no Admin section at all. The section is not rendered, not greyed out.
