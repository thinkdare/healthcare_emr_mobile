Implementation notes

  • Two API calls on load: GET /organizations/{auth.organizationId} + GET /organizations/{id}/stats (parallel).
  • State: _isEditing bool — pencil icon toggles it. Cancel reverts controllers to original values. Save calls PUT /organizations/{id}.
  • New OrganizationRepository methods: getOrganization(id), getOrgStats(id), updateOrganization(id, data).
  • No new provider needed — the screen manages its own state (StatefulWidget) since org profile is a single-use screen not shared across the app.
  • Type dropdown values: hospital, clinic, pharmacy, laboratory, diagnostic_center, hospital_group, other.
